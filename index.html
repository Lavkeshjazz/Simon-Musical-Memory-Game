Rules for Playing the Game

Rule 1

Press the START button. Simon will give the first signal. Repeat the signal by pressing the same character lens.

Rule 2

Simon will duplicate the first signal and add one. Repeat these two signals by pressing the same character lenses, in order.

Rule 3

Simon will duplicate these first two signals and add one.

Rule 4

Continue playing as long as you can repeat each sequence of siginals correctly.

Rule 5

If you fail to repeat a sequence exactly, Simon responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends.

Rule 6

HAVE FUN! and try again again to beat your highscore.
